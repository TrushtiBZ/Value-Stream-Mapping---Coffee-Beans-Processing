# Refining Coffee Processing - Value Stream Mapping (Lean Methodology)

## Overview

This project focuses on analyzing and improving the operational efficiency of a refining coffee processing by applying Lean Manufacturing principles, specifically Value Stream Mapping (VSM).

The goal of this analysis was to identify waste, delays, and bottlenecks in the production flow that impact the company’s ability to meet customer demand. By mapping the current state of the production process, I evaluated value-added vs non-value-added activities, calculated key performance metrics, and identified improvement opportunities.

## Objectives

**The main objectives of this project were to:**

- Analyze the end-to-end coffee beans processing workflow

- Identify process inefficiencies and operational bottlenecks

- Distinguish between Value-Added (VA) and Non-Value-Added (NVA) activities

- Calculate key Lean performance metrics

- Evaluate the company’s ability to meet customer demand

- Provide insights for process improvement and capacity planning

## Process Flow Analyzed

**The value stream map was developed for the following coffee beans processing stages:**

- Hulling

- Cleaning

- Grading

- Roasting

- Packaging

- Shipping

**Each stage was analyzed for:**

- Cycle Time

- Setup Time

- Processing Time

- Equipment Utilization

- Units produced per day

 ## Methodology Used

This project uses Lean Manufacturing – Value Stream Mapping (VSM) to visualize and analyze the production flow.

**Steps performed in the analysis:**

- Mapped the current state value stream

- Calculated process data metrics

- Measured cycle time and throughput

- Identified inventory buildup between processes

- Determined bottlenecks and constraints

- Evaluated process cycle efficiency

## Key Metrics Calculated
- Takt Time - 0.45 seconds per unit - This represents the rate at which the company must produce coffee beans to meet customer demand.

- Total Production Lead Time - 9.5 days - This includes the total time required from receiving the order to delivering the finished product.

- Total Cycle Time - 535 seconds - This represents the total processing time required across all manufacturing stages.

- Weekly Production Capacity - 1,350 units per week - This capacity is based on the current system constraints and machine utilization.

## Bottleneck Identification

The analysis identified Hulling as the primary bottleneck in the production line.

**Because of its:**

- Higher cycle time

- Lower processing throughput

- This stage limits the system's ability to meet customer demand requirements.

**Process Cycle Efficiency (PCE)**

PCE = 0.104% - This indicates that only a very small portion of the total production lead time is spent on value-adding activities, highlighting significant opportunities for process improvement and waste reduction.

## Key Insights

Through the VSM analysis, the following operational insights were identified:

- inventory waiting times between production stages

- Imbalance in processing capacities across departments

- Underutilization in some processes while others create bottlenecks

- Excessive lead time compared to actual processing time

- Opportunities for workflow balancing and process redesign

## Improvement Opportunities

Based on Lean principles, the following improvements can help optimize the process:

- Reduce setup times in critical processes

- Balance production capacity across stages

- Implement pull-based production flow

- Improve equipment utilization

- Reduce intermediate inventory buffers

- These improvements could significantly increase throughput, efficiency, and responsiveness to customer demand.
